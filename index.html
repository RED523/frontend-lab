Debounce 防抖函数演示
这个示例演示了 JavaScript 中防抖（Debounce）函数的实现和应用场景。
  • **防抖原理**：在事件被触发 n 秒后再执行回调，如果在这 n 秒内又被触发，则重新计时
  • **应用场景**：搜索框输入、按钮点击、窗口大小调整等高频事件
  • **与节流的区别**：防抖是延迟执行，节流是限制执行频率
